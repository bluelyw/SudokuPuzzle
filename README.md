# 🎮 趣味数独游戏

一个专为6-12岁小朋友设计的数独游戏，界面友好，操作简单！

## 🌟 游戏特色

- **多种宫格大小**：4宫格、6宫格、9宫格、16宫格，适合不同年龄段
- **适合儿童**：大按钮、鲜艳颜色、简单操作
- **三种难度**：简单、中等、困难，适合不同水平
- **智能提示**：最多3次提示机会，帮助小朋友学习
- **实时计时**：记录游戏时间，增加挑战性
- **即时反馈**：错误提示和成功动画
- **响应式设计**：支持手机和平板设备

## 🎯 游戏规则

数独是一个数字填充游戏，需要填入数字，使得：

1. 每一行都包含所有数字，不重复
2. 每一列都包含所有数字，不重复  
3. 每个宫格都包含所有数字，不重复

### 不同宫格大小说明

- **4宫格**：2×2宫格，数字1-4，适合6-7岁小朋友
- **6宫格**：2×3宫格，数字1-6，适合7-8岁小朋友
- **9宫格**：3×3宫格，数字1-9，适合8-10岁小朋友
- **16宫格**：4×4宫格，数字1-16，适合10-12岁小朋友

## 🎮 操作方法

### 基本操作
- **选择单元格**：点击数独网格中的任意空白单元格
- **填入数字**：点击数字按钮或使用键盘数字键
- **清除数字**：点击"🗑️ 清除"按钮或使用删除键

### 游戏功能
- **🆕 新游戏**：开始新的数独游戏
- **✅ 检查**：检查当前解答是否正确
- **💡 提示**：获得一个正确答案的提示（最多3次）
- **🔍 解答**：自动完成整个数独

### 难度选择
- **简单**：保留50%的数字，适合初学者
- **中等**：保留40%的数字，有一定挑战性
- **困难**：保留30%的数字，适合熟练玩家

### 宫格大小选择
- **4宫格**：最简单的数独，数字1-4
- **6宫格**：适合初学者的数独，数字1-6
- **9宫格**：经典数独，数字1-9
- **16宫格**：高级数独，数字1-16

## 🚀 如何开始

1. 在浏览器中打开 `index.html` 文件
2. 选择适合的宫格大小和难度级别
3. 点击空白单元格，然后填入数字
4. 完成所有单元格后点击"检查"验证答案
